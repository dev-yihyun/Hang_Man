## [프로젝트 소개]
- <u>개인 프로젝트</u>
- 자바스크립트를 공부하면서 기술을 익히기 위해 진행한 프로젝트입니다.
- 알파벳 버튼을 클릭하면 해당 단어에 존재할 경우 빈칸을 채우고 존재하지 않을 경우 이미지가 나타나며 라이프를 깎습니다.
